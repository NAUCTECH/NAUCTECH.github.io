Scratch Jr. Programming

Grade Levels: 2-4: The goal of this course is to build the foundation for the thought processes that are necessary for all future coding. We will use Scratch jr, a free block programming app developed in collaboration between the DevTech Research Group at Tufts University, the Lifelong Kindergarten Group at the MIT Media Lab, and the Playful Invention Company. Block programming will introduce the core computing concepts of space, motion, variables, conditionals, functions, objects, loops, and a variety of mathematical concepts through drag and drop commands.



Scratch Jr. Programming

Grade Levels: 5-8: The goal of this course is to build the foundation for the thought processes that are necessary for all future coding. We will use Scratch, a free block programming language developed by MIT, to teach the core concepts of programming. Block programming will introduce the core computing concepts of space, motion, variables, conditionals, functions, objects, loops, and a variety of mathematical concepts through drag and drop commands.



Dash and Dot Robotics

Grade Levels: 2-4: The goal of this course is to allow coders to use their knowledge of computer science to interact with the world around them. Dash and Dot robots are programmed through a block programing language that coders can utilize based on their prior knowledge of Scratch jr. The basics of hardware and hardware/software interaction will be introduced.



Lego Mindstorm Robotics

Grade Levels: 5-8: LegoR has created an in-depth, entertaining way for coders to experiment with intermediate level robotics. This class will invite coders to be creative with their robot design while still using block programming. Coders will be introduced to all of the hardware connections before interacting with the programming software.
